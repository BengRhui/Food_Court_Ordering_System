# OODJ ASSIGNMENT

Hi everyone! This is the repository for our Java group project.

---

## Introduction
This Java project is about a food court management system, featuring 5 different users with each assigned with distinct reponsibilities.

---

## What we learnt?
Our project has demonstrated a few programming concepts:
- Object-oriented programming concepts (*encapsulation, inheritance, abstraction, polymorphism*)
- Test plans
- GUI components (*via Java Swing and AWT components*)

---

## Running Our Project
To run our program, you shall make sure that:
- Java Development Kit (JDK) is installed on your computer (*preferably later than JDK 21*).
- (*Optional*) An IDE to run Java prorgams, e.g. IntelliJ IDEA / Netbeans / Visual Studio Code.
 
All credentials for login can be found at `\this\path\to\be\updated\later`.
